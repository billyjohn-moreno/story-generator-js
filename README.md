# Story Generator (JavaScript)

Interactive story generator built with vanilla JavaScript. Users can select different genres and display dynamic stories.

## Features

- Dynamic story display
- Multiple genres:
  - Scary
  - Funny
  - Adventure

- DOM manipulation
- Event-driven interaction
- Dynamic UI styling

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Concepts Practiced

- DOM selection
- Event listeners
- Objects and properties
- Functions and parameters
- Conditional logic
- Dynamic style manipulation

## Learning Purpose

This project was created to practice JavaScript fundamentals before transitioning to React.

## Future Improvements

- Convert to React
